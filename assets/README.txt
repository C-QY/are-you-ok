Easter Egg Audio
================

Place "leijun.mp3" (or "leijun.wav") in this folder to enable the audio Easter egg.
The clip plays when the skill is triggered by the exact phrase "are you ok".

Where to get it:
  Search "雷军 are you ok" on Bilibili or YouTube.
  Trim the clip to ~3 seconds and save as leijun.mp3.

If the file is absent, a text Easter egg is shown instead — the skill works normally either way.

This folder is intentionally empty in the repository (no audio distributed due to copyright).
