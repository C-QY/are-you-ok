Easter Egg Audio
================

Five Lei Jun audio clips (2015 Shanghai speech) — all play silently in background, no window.

  File                               Trigger phrase                  Duration
  ─────────────────────────────────────────────────────────────────────────────
  eleijun-are-you-ok.mp3             "are you ok"                    ~2.3s
  eleijun-hello.mp3                  "hello"                         ~1.5s
  eleijun-thank-you.mp3              "thank you"                     ~1.4s
  eleijun-thank-you-very-much.mp3    "thank you very much"           ~2.1s
  eleijun-super.mp3                  all 4 phrases in one message    ~18s

WAV variants (eleijun-<name>.wav) are also supported as fallback.

Source: https://sc.chinaz.com/yinxiao/180322325970.htm  (18s, free for personal use)
Encode: 64kbps mono 22050Hz MP3 (voice quality, minimal size)

If a file is absent, a text easter egg box is shown instead — skill works normally either way.
No audio pre-built in repository (copyright).
